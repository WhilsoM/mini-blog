# mini blog on React

I use in this project React + Ts + Tailwind

## Routes

| Путь          | Компонент | Описание                    |
| ------------- | --------- | --------------------------- |
| `/`           | Home      | Главная: список всех постов |
| `/post/:slug` | Post      | Отдельный пост              |
| `/profile`    | Profile   | Информация о себе           |
